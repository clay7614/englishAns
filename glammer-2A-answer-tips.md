# glammer-2A文法テスト 正答の見分け方メモ

glammer-2A.csvの設問に共通する選択肢のクセをまとめています。迷ったときのチェックリストとして使ってください。

- 動名詞/不定詞の相性: enjoy / avoid / give up / mind / stop / finish / go on → 動名詞。decide / want / hope / promise / plan / expect / learn → to不定詞。tryは「試しに」は動名詞、「努力して」はto不定詞。stopは「〜をやめる」なら動名詞、「立ち止まって〜する」ならto不定詞。
- 目的・結果のto不定詞: in order to / so as toで目的。「only to」「enough to」「too 〜 to」「decision/plan/promise + to」で結果や意図を表すパターンが多い。
- it 〜 to構文: It is + 形容詞 + (for/of) 人 + to V で形式主語を立てる問題が頻出。
- 使役・知覚動詞: make / let / have + O + 動詞原形、get + O + to不定詞。hear / see / watch / feel + O + 動詞原形/現在分詞/過去分詞。keep / leave / find + O + 形容詞/分詞もよく問われる。
- 分詞構文: 主語が同じときに文頭を -ing / Not -ing / Having + 過去分詞 で繋ぐ。Having + p.p. は「先に済ませて/したので」の意味。
- 前置詞＋動名詞: be proud of / be afraid of / be used to / look forward to / give up / avoid など前置詞や動詞の後は -ing を選ぶ。
- 形容詞の -ing / -ed: -ing は「〜させる側」「ものごと」、-ed は「〜される側」「人の感情」。boring game / bored people など。
- 比較表現: as 〜 as / not as 〜 as、比較級 + than、much / far / a lot + 比較級、twice / three times as 〜 as、the 〜er ... the 〜er、any other + 単数で「どの〜よりも」、of the two には the + 比較級。
- 名詞を修飾する分詞: 現在分詞で能動・進行 (running water)、過去分詞で受動・完了 (broken glass)。前置詞句の後ろに置いて自然な語順を確認する。
- 目的語と補語を伴う動詞: keep / leave / find / make / get などは O + 形容詞/分詞を後ろに取り、自然な語感を選ぶ（keep the machine running, find it difficult to ...）。
